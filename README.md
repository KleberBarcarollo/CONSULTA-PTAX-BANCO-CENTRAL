# CONSULTA-PTAX-BANCO-CENTRAL
CONSULTA PTAX BANCO CENTRAL
Realizar a consulta da cotação PTAX do Dolar direto do site do Banco Central

Sobre este fluxo de automação:
Este robô pode ir até o site do Banco Central, preencher este formulário com a data de ontem e coletar a tabela do dólar PTAX para ser utilizado para atualizar outros sistemas com essa informação.
